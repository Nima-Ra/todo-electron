# Hot keys
| Hot Key       | Mac OS        |Linux / Windows  |
| ------------- |:-------------:| ---------------:|
| Add todo      | Command + T   | Ctrl + T        |
| Clear todos   | Command + K   | Ctrl + K        |
| Quit          | Command + Q   | Ctrl + Q        |

# How to launch it?
1. Extract the .tar.xz file
2. If you use linux, launch **Todo** file 
3. If you use windows, launch **Todo.exe**
